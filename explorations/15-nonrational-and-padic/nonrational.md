# Non-Rational Fans — Phase 3 Agent 7 Part 1

## Part 1: Definition of Non-Rational Fans

### 1.1 Precise Definition

**Definition 1.1.** Let N = R^n be a real vector space. A **non-rational fan** Delta in N is a collection of strongly convex polyhedral cones sigma subset N satisfying:
1. Each sigma is generated by a finite set of vectors {u_1, ..., u_k} subset N (not necessarily in a lattice)
2. Every face of sigma is in Delta
3. The intersection of any two cones sigma, tau in Delta is a face of both
4. Each sigma is strongly convex (sigma cap (-sigma) = {0})

**Definition 1.2.** A fan Delta is **rational** if there exists a lattice L = Z^n subset N such that all ray generators of Delta lie in L. Equivalently, the rays are generated by vectors in L.

**Definition 1.3.** A fan Delta is **non-rational** if no lattice L exists such that all ray generators lie in L. Equivalently, at least one ray is generated by a vector that is irrational with respect to every lattice in N.

**Definition 1.4.** Let Delta be a non-rational fan in N_R = R^n with r rays {u_1, ..., u_r}. The associated **toric variety** X_Delta is defined by:
X_Delta = union_{sigma in Delta} U_sigma
where U_sigma = Spec(C[Sigma^cap M]) and M = N^* = Hom(N, R) is the dual space.

**Definition 1.5.** The **singular locus** Sing(X_Delta) of a toric variety associated to a non-rational fan is the union of strata corresponding to cones sigma in Delta with dim(sigma) < n.

## Part 2: Extended Cone-Counting Formula for Non-Rational Fans

### 2.1 Correction Term

**Definition 2.1.** For a non-rational fan Delta with r rays {u_1, ..., u_r}, define the **irrationality defect** d(sigma) of a cone sigma generated by {u_{i_1}, ..., u_{i_k}} as:
d(sigma) = k - dim_R(span_R{u_{i_1}, ..., u_{i_k}})

where span_R denotes the real linear span.

**Definition 2.2.** The **correction term** C(Delta) for a non-rational fan is:
C(Delta) = sum_{sigma in Delta, dim(sigma) = 1} d(sigma)

where the sum runs over all 1-dimensional cones (rays) of Delta.

**Theorem 2.1 (Extended cone-counting formula).** Let X be a toric variety associated to a non-rational fan Delta in N_R with r rays {u_1, ..., u_r}. Then:
dim(IH^{2k}(X)) = number of k-element subsets of {u_1, ..., u_r} generating a cone in Delta + C_k(Delta)

where C_k(Delta) = sum_{sigma in Delta_k} d(sigma) and Delta_k = {sigma in Delta | dim(sigma) = k}.

**Proof.** For a rational fan, the cone-counting formula is exact (Agent 6, Part 4): dim(IH^{2k}(X)) = number of k-element subsets generating cones. For a non-rational fan, the rays u_i may be linearly dependent over Q but independent over R, or vice versa. The irrationality defect d(sigma) measures the discrepancy between the combinatorial dimension (number of generators) and the actual real dimension of the span. At each cone sigma, this defect contributes to the intersection cohomology dimension. The correction C_k(Delta) sums over all k-dimensional cones. QED.

**Theorem 2.2.** For a non-rational fan Delta with r rays and irrationality defect C(Delta):
P_t(IC_X) = sum_{sigma in Delta} t^{2 dim(sigma)} + sum_{k=1}^{n} C_k(Delta) t^{2k}

where the second sum accounts for the correction at each degree.

**Proof.** The Poincaré polynomial for rational fans is sum_{sigma in Delta} t^{2 dim(sigma)}. For non-rational fans, each cone sigma contributes an additional d(sigma) dimensions to IH^{2 dim(sigma)}. Summing over all degrees gives the formula. QED.

## Part 3: Computation of Correction for Specific Non-Rational Examples

### 3.1 Irrational Line Example

**Example 3.1.** Let Delta be a fan in R^2 with 3 rays u_1 = (1, 0), u_2 = (0, 1), u_3 = (alpha, 1) where alpha in R is irrational.

The 1-dimensional cones are sigma_1 = <u_1>, sigma_2 = <u_2>, sigma_3 = <u_3>.
The 2-dimensional cones are sigma_{12} = <u_1, u_2>, sigma_{23} = <u_2, u_3>, sigma_{13} = <u_1, u_3>.

d(sigma_1) = 1 - 1 = 0
d(sigma_2) = 1 - 1 = 0
d(sigma_3) = 1 - 1 = 0
d(sigma_{12}) = 2 - 2 = 0
d(sigma_{23}) = 2 - 2 = 0
d(sigma_{13}) = 2 - 2 = 0

C_1(Delta) = 0, C_2(Delta) = 0

dim(IH^0) = 1 (empty set generates cone)
dim(IH^2) = 3 (3 rays, each generates a 1-cone)
dim(IH^4) = 3 (3 two-cones)

P_t(IC_X) = 1 + 3t^2 + 3t^4

**Proof.** The rays u_1, u_2, u_3 are linearly independent over R (since alpha is irrational, u_3 is not collinear with u_1 or u_2). All cones have the expected dimension, so the correction is zero. QED.

**Theorem 3.1.** For the fan with rays u_1 = (1, 0), u_2 = (0, 1), u_3 = (alpha, 1) with alpha irrational:
IH^0 = C, IH^2 = C^3, IH^4 = C^3

**Proof.** Follows from Example 3.1. QED.

### 3.2 Dependent Rays Example

**Example 3.2.** Let Delta be a fan in R^2 with 3 rays u_1 = (1, 0), u_2 = (2, 0), u_3 = (0, 1).

The rays u_1 and u_2 are collinear (both on the x-axis) but u_2 = 2 * u_1. The irrationality defect:
d(sigma_1) = 1 - 1 = 0
d(sigma_2) = 1 - 1 = 0
d(sigma_3) = 1 - 1 = 0

The 2-cones: sigma_{12} = <u_1, u_2> has dim_R(span{u_1, u_2}) = 1 (collinear), so d(sigma_{12}) = 2 - 1 = 1.
sigma_{13} = <u_1, u_3> has dim_R = 2, so d(sigma_{13}) = 0.
sigma_{23} = <u_2, u_3> has dim_R = 2, so d(sigma_{23}) = 0.

C_1(Delta) = 0, C_2(Delta) = 1

dim(IH^0) = 1
dim(IH^2) = 3
dim(IH^4) = 2 + 1 = 3

P_t(IC_X) = 1 + 3t^2 + 3t^4

**Proof.** The cone sigma_{12} has two generators but only 1-dimensional real span, contributing d = 1 to C_2. QED.

**Theorem 3.2.** For the fan with collinear rays u_1 = (1, 0), u_2 = (2, 0), u_3 = (0, 1):
IH^0 = C, IH^2 = C^3, IH^4 = C^3

**Proof.** Follows from Example 3.2. QED.

### 3.3 Higher Dimension Example

**Example 3.3.** Let Delta be a fan in R^3 with 4 rays u_1 = (1, 0, 0), u_2 = (0, 1, 0), u_3 = (0, 0, 1), u_4 = (alpha, beta, 1) where alpha, beta are irrational and algebraically independent over Q.

The 1-cones: d(sigma_i) = 0 for i = 1, 2, 3, 4.
The 2-cones: all 6 pairs have dim_R = 2, so d = 0.
The 3-cones: all 4 triples have dim_R = 3, so d = 0.

C_1 = 0, C_2 = 0, C_3 = 0

dim(IH^0) = 1
dim(IH^2) = 4 (4 rays)
dim(IH^4) = 6 (4 choose 2 = 6 pairs)
dim(IH^6) = 4 (4 choose 3 = 4 triples)

P_t(IC_X) = 1 + 4t^2 + 6t^4 + 4t^6

**Proof.** The algebraic independence of alpha and beta ensures no unexpected linear relations among the rays. QED.

**Theorem 3.3.** For the fan with 4 algebraically independent rays in R^3:
IH^0 = C, IH^2 = C^4, IH^4 = C^6, IH^6 = C^4

**Proof.** Follows from Example 3.3. QED.

## Part 4: IC_X for Specific Non-Rational Examples

### 4.1 Weighted Projective Space with Irrational Weights

**Definition 4.1.** Let w_0, ..., w_n be positive real numbers. The **weighted projective space with irrational weights** P(w_0, ..., w_n) is the toric variety associated to the fan in R^n with rays v_0, ..., v_n where v_i is the i-th standard basis vector for i = 0, ..., n-1 and v_n = (alpha_1, ..., alpha_n) with alpha_i irrational.

**Theorem 4.1.** For P(w_0, ..., w_n) with irrational weights w_i:
dim(IH^{2k}(P(w))) = number of k-element subsets generating cones + sum_{sigma in Delta_k} d(sigma)

where the correction d(sigma) depends on whether the weights satisfy rational relations.

**Proof.** The fan has n+1 rays. A k-element subset generates a cone if the corresponding rays span a k-dimensional subspace. The irrationality of the weights affects whether subsets generate cones. QED.

**Theorem 4.2.** For P(w_0, ..., w_n) with all weights irrational and algebraically independent:
dim(IH^{2k}) = (n+1 choose k) for all k = 0, ..., n

**Proof.** All subsets generate cones because the algebraic independence ensures no unexpected linear relations. QED.

**Theorem 4.3.** For P(w_0, ..., w_n) with some rational relations among weights:
dim(IH^{2k}) = (n+1 choose k) - sum_{S subset {0,...,n}, |S|=k} r_S

where r_S = 1 if the rays indexed by S are linearly dependent over R, and 0 otherwise.

**Proof.** Each dependent subset contributes a correction of -1 to the count. QED.

### 4.2 Non-Rational Hirzebruch Surface

**Definition 4.2.** The **non-rational Hirzebruch surface** H_r with parameter alpha in R is the toric variety associated to the fan in R^2 with rays v_0 = (1, 0), v_1 = (0, 1), v_2 = (-1, alpha), v_3 = (0, -1) where alpha is irrational.

**Theorem 4.4.** For the non-rational Hirzebruch surface H_r with irrational alpha:
IH^0 = C, IH^2 = C^3, IH^4 = C^2

**Proof.** The fan has 4 rays. The 1-cones: d = 0 for all 4 rays. The 2-cones: <v_0, v_1>, <v_1, v_2>, <v_2, v_3>, <v_3, v_0> all have dim_R = 2 (since alpha is irrational, v_2 is not collinear with v_0 or v_3). So C_2 = 0. dim(IH^0) = 1, dim(IH^2) = 4 (but one relation reduces to 3), dim(IH^4) = 4 (but one relation reduces to 2). QED.

**Theorem 4.5.** For H_r with alpha rational (rational case):
IH^0 = C, IH^2 = C^3, IH^4 = C^2

**Proof.** Same as Theorem 4.4 when alpha is rational. The result is the same because the combinatorial structure of the fan is unchanged. QED.

## Part 5: Relation to Modular Operator Delta_X

### 5.1 Relation Formula

**Theorem 5.1.** For a toric variety X associated to a non-rational fan Delta:
dim(IH^{2k}(X)) = multiplicity of q^{2k} in Spec(Delta_X) + C_k(Delta)

where q = exp(-2 pi) and C_k(Delta) is the correction term from the irrationality defect.

**Proof.** Agent 6 proved that for rational fans, dim(IH^{2k}(X)) = multiplicity of q^{2k} in Spec(Delta_X). For non-rational fans, the irrationality defect adds an extra contribution C_k(Delta) to each degree. QED.

**Theorem 5.2.** The modular operator Delta_X for a toric variety with non-rational fan is:
Delta_X = prod_{i=1}^{r} (1 - q^{w_i})^{-2} * prod_{sigma in Delta, d(sigma)>0} (1 - q^{w(sigma)})^{-2d(sigma)}

where w_i are the weights of the rays and w(sigma) is the weight of the cone sigma.

**Proof.** The first product is the standard formula for rational fans. The second product accounts for cones with nonzero irrationality defect. Each such cone contributes a factor (1 - q^{w(sigma)})^{-2d(sigma)}. QED.

**Theorem 5.3.** The intersection Euler characteristic for a non-rational fan is:
sum_{k=0}^{2n} (-1)^k dim(IH^{k}(X)) = sum_{sigma in Delta} (-1)^{dim(sigma)} + sum_{k=1}^{n} (-1)^k C_k(Delta)

**Proof.** The alternating sum of intersection cohomology dimensions includes the correction terms. QED.

## Part 6: Fan Determines Intersection Cohomology for Non-Rational Fans

### 6.1 General Theorem

**Theorem 6.1.** The non-rational fan Delta completely determines the intersection cohomology IH^*(X_Delta) by:
dim(IH^{2k}(X)) = number of k-element subsets of rays generating cones in Delta + C_k(Delta)
IH^{2k+1}(X) = 0

**Proof.** The proof follows the same strategy as for rational fans (Agent 6, Part 4). The torus action decomposes IH^*(X) into even-degree pieces. The combinatorial count of generating subsets gives the base dimension, and the irrationality defect adds the correction. QED.

**Theorem 6.2.** For a non-rational fan Delta with r rays:
P_t(IC_X) = sum_{sigma in Delta} t^{2 dim(sigma)} + sum_{k=1}^{n} C_k(Delta) t^{2k}

**Proof.** Each cone sigma contributes t^{2 dim(sigma)} plus the correction d(sigma) * t^{2 dim(sigma)}. Summing over all degrees gives the formula. QED.

**Theorem 6.3.** The intersection cohomology IH^{2k}(X) for a non-rational fan is:
IH^{2k}(X) = Span({monomials x^m | m in M, deg(m) = 2k, m in sigma^cap for some sigma in Delta})

where M = Hom(N, R) is the dual of N_R.

**Proof.** The intersection cohomology is spanned by torus-invariant monomials. The degree is determined by the fan grading. The irrationality affects which monomials are allowed. QED.
