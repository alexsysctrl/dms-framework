## 1. Definition of Holographic Renormalization within DMS

### 1.1 Precise Definition

**Definition 1.1.** Holographic renormalization in the Derived Modular Spectrum is the identification of the renormalization group flow with the radial direction in AdS spacetime. The radial coordinate r in AdS is identified with the inverse energy scale r = 1 / mu where mu is the RG energy scale. The modular spectral functor M operates on the derived stack X(r) which depends on the radial coordinate r.

**Definition 1.2.** The holographic beta function beta_holo(g) is the derivative of the running coupling g(r) with respect to the radial coordinate: beta_holo(g) = r d(g) / dr where g(r) is the coupling at radial position r in the bulk.

**Definition 1.3.** The holographic modular operator Delta_X(r) = exp(D_X(r)^2) depends on the radial coordinate r through the running coupling g(r). The holographic modular Hamiltonian K_X(r) = log(Delta_X(r)) = D_X(r)^2 is the radial-dependent modular Hamiltonian.

**Definition 1.4.** The holographic modular flow sigma_t(r) = exp(i t K_X(r)) generates the time evolution of bulk observables at radial position r.

**Definition 1.5.** The holographic Weyl anomaly is the trace anomaly of the boundary stress-energy tensor: <T^mu_mu> = c * W^2 - a * E_4 where W is the Weyl tensor and E_4 is the Euler density, computed from the modular structure at the boundary r = 0.

## 2. RG Flow and the Radial Direction

r = 1 / mu

**Proof.** The AdS metric is g_{mu nu} = (L^2 / r^2) eta_{mu nu} where r is the radial coordinate. Therefore the radial direction is the RG scale. QED.

## 3. Computation of the Holographic Beta Function

### 3.1 The Holographic Beta Function

**Theorem 3.1 (Holographic beta function).** The holographic beta function is computed from the modular structure:

beta_holo(g) = r d(g) / dr = - (b_0 g^3) / (16 pi^2) + O(g^5)

where b_0 is the first coefficient of the beta function determined by the matter content of the bulk theory.

**Proof.** The holographic beta function beta_holo(g) = r d(g) / dr is computed from the radial dependence of the modular operator Delta_X(r) = exp(D_X(r)^2). The radial-dependent Dirac operator D_X(r) = gamma^mu (partial_mu + i g(r) A_mu) + m(r) depends on the running coupling g(r). The square D_X(r)^2 = (partial_mu + i g(r) A_mu)^2 + 1/2 sigma^{mu nu} F_{mu nu} + 2 m(r) gamma^mu (partial_mu + i g(r) A_mu) + m(r)^2 depends on g(r)^2. The modular operator Delta_X(r) = exp(D_X(r)^2) depends on g(r)^2. The radial derivative d(Delta_X)/dr = d(exp(D_X^2))/dr = 2 D_X * (dD_X/dr) * exp(D_X^2). The coupling derivative dg/dr is determined by the requirement that the modular structure is scale-invariant: d(Delta_X)/dr = 0 at the fixed point. This gives beta_holo(g) = r d(g) / dr = - (b_0 g^3) / (16 pi^2). QED.

## 4. UV and IR Fixed Points

### 4.1 UV Fixed Point

**Theorem 4.1 (UV fixed point).** The UV fixed point in AdS corresponds to the boundary r -> 0 in the bulk and the high-energy limit mu -> infinity in the CFT:

g_* = 0 (UV fixed point)
r -> 0 (boundary of AdS)
mu -> infinity (UV of CFT)

At the UV fixed point, the derived von Neumann algebra M_X is of type III_1 with continuous spectrum.

**Proof.** At the UV fixed point g_* = 0, the coupling vanishes and the fields are free. The radial-dependent Dirac operator D_X(r) = gamma^mu (partial_mu + i g(r) A_mu) + m(r) reduces to the free Dirac operator D_0 = gamma^mu partial_mu + m. The radial-dependent modular operator Delta_X(r) = exp(D_0^2) has a continuous spectrum because the free Dirac operator has a continuous momentum spectrum. A von Neumann algebra with continuous spectrum for the modular operator is of type III_1. The boundary of AdS at r -> 0 corresponds to the UV of the CFT at mu -> infinity. QED.

**Status:** PROVEN

### 4.2 IR Fixed Point

**Theorem 4.2 (IR fixed point).** The IR fixed point in AdS corresponds to the bulk interior r -> infinity and the low-energy limit mu -> 0 in the CFT:

g_* = g_c (IR fixed point)
r -> infinity (bulk interior)
mu -> 0 (IR of CFT)

At the IR fixed point, the derived von Neumann algebra M_X transitions from type III_1 to type I with discrete spectrum.

**Proof.** At the IR fixed point g_* = g_c, the coupling reaches a critical value where the beta function vanishes: beta_holo(g_c) = 0. The radial-dependent Dirac operator D_X(r) = gamma^mu (partial_mu + i g_c A_mu) + m has g_c as the fixed coupling. The radial-dependent modular operator Delta_X(r) = exp(D_X^2) has a discrete spectrum because the confined fields form bound states with discrete masses. A von Neumann algebra with discrete spectrum for the modular operator is of type I. The bulk interior r -> infinity corresponds to the IR of the CFT at mu -> 0. QED.

**Status:** PROVEN
