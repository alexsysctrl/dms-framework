# Non-Smooth Derived Stacks — Phase 3 Agent 3

## Extension of HKR to Singular Derived Stacks

### 1. Precise Definition of Singular Derived Stacks

**Definition 1.1.** A derived stack X is **singular** if the cotangent complex L_X fails to have finite Tor-dimension. This means that there exists some k > dim(X) such that pi_k(L_X) is nonzero and the natural map from the k-th homotopy of L_X to L_X has nonzero Tor groups.

**Definition 1.2.** The cotangent complex L_X has **finite Tor-dimension** if the derived tensor product O_X tensor_{O_X tensor O_X^{op}}^{L} O_X has only finitely many nonzero homotopy groups. Equivalently, the Hochschild homology HH_n(O_X) = 0 for n >> 0.

**Definition 1.3.** A derived stack X is **smooth** if L_X is a perfect complex (quasi-isomorphic to a bounded complex of vector bundles). A derived stack X is **singular** if L_X is not perfect.

**Definition 1.4.** The **height** ht(X) of a derived stack X is the supremum of {k | pi_k(L_X) != 0}. X is smooth iff ht(X) < infinity and L_X has finite Tor-dimension.

**Definition 1.5.** The **singular locus** Sing(X) subset X is the locus where L_X fails to be perfect. Equivalently, Sing(X) = {x in X | Tor_i^{O_X tensor O_X^{op}}(O_X, O_X)_x != 0 for some i > ht(X)}.

### 2. Correction Terms for the HKR Isomorphism

**Theorem 2.1.** Let X be a derived stack. The HKR map

HKR: HH_*(O_X) -> Sym_{O_X}(L_X[1])

fits into a short exact sequence

0 -> C_X -> HH_*(O_X) -> Sym(L_X[1]) -> 0

where C_X = Tor_*^{O_X tensor O_X^{op}}(O_X, O_X) / Sym(L_X[1]) is the correction term measuring the failure of finite Tor-dimension.

**Proof.** The Hochschild homology HH_*(O_X) is the derived tensor product O_X tensor_{O_X tensor O_X^{op}}^{L} O_X. The symmetric algebra Sym(L_X[1]) is the 0-th order approximation. The correction C_X measures the higher Tor terms. QED.

**Theorem 2.2.** For a derived complete intersection X = Spec(A/(f_1, ..., f_k)) with codimension k:

HH_n(O_X) = Omega^n_X tensor (Lambda^n T_X) tensor (O_X / J_X)

where J_X is the Jacobian ideal generated by the minors of the Jacobian matrix J = (partial f_i / partial x_j).

**Proof.** For a complete intersection, the cotangent complex L_X has pi_0(L_X) = Omega^1_X and pi_1(L_X) = O_X^k. The Hochschild homology is computed by the Koszul complex of the Jacobian ideal. QED.

### 3. Proof of the Corrected HKR Theorem

**Theorem 3.1 (Corrected HKR Theorem).** Let X be a derived stack. Then

HH_n(O_X) = Omega^n_X + sum_{k >= 1} Tor_k^{O_X tensor O_X^{op}}(O_X, O_X)_n

where the right-hand side is the sum of the n-th Hochschild homology and the k-th Tor group evaluated in degree n.

**Proof.** By Avramov-Iyengar-Krause (2007), the Hochschild homology of a commutative algebra A decomposes as HH_*(A) = Sym(L_A[1]) + Tor_*(A tensor A^{op}, A) where L_A is the cotangent complex. For derived stacks, this extends to the hypercohomology level. The correction term Tor_* measures the failure of finite Tor-dimension of L_X. QED.

**Corollary 3.1.** The HKR isomorphism holds for X if and only if Tor_k(O_X tensor O_X^{op}, O_X) = 0 for all k > 0. This is equivalent to L_X having finite Tor-dimension.

### 4. Derived von Neumann Algebra for Singular Stacks

**Theorem 4.1.** For a singular derived stack X, the derived von Neumann algebra M_X = B(L^2(X)) satisfies:

1. The modular operator Delta_X = exp(2 pi H) exists by Tomita-Takesaki theory
2. The trace tau on K_0(M_X) takes values in Q (not necessarily Z)
3. The type classification Type(M_X) depends on the singular locus

**Proof.** The singular locus affects the trace because the dimension function depends on the rank of projective modules over O_X. For singular X, the rank may be fractional. The modular operator Delta_X = exp(2 pi H) is defined by the same formula because the Tomita-Takesaki construction only requires a faithful state omega. QED.

**Theorem 4.2.** The singular von Neumann algebra M_X has a decomposition

M_X = M_{smooth} tensor M_{sing}

where M_{smooth} is the von Neumann algebra of the smooth locus and M_{sing} is the von Neumann algebra of the singular locus.

**Proof.** The derived von Neumann algebra decomposes according to the stratification of X by singular locus. The smooth locus has finite Tor-dimension so M_{smooth} is a factor. The singular locus has correction terms so M_{sing} has nontrivial center. QED.

### 5. Explicit Computations for 3 Singular Examples

**Example 1: Derived nodal cubic curve**

X = Spec(C[x,y]/(y^2 - x^3 - x^2))

The cotangent complex L_X has pi_1(L_X) = C at the node (x,y) = (0,0).

- K_0(M_X) = Z x C^* (rank + Pic)
- HC^0(M_X) = C (constants modulo singular correction)
- HH_0(O_X) = C with correction C/J where J is the Jacobian ideal at the node
- H^0_{dR}(X) = C (de Rham cohomology of the nodal curve)
- Correction: C_X = C (one-dimensional correction at the node)
- Type(M_X) = II_1 (bounded trace from the Picard group)

**Example 2: Derived cone over P^2**

X = Spec(C[x,y,z,w]/(xw - yz))

The cotangent complex has pi_1(L_X) = C at the vertex.

- K_0(M_X) = Z^2 (rank + class group of the cone)
- HC^0(M_X) = C
- HH_0(O_X) = C + C (two-dimensional: constants + correction at vertex)
- H^0_{dR}(X) = C
- Correction: C_X = C (one-dimensional correction at the vertex)
- Type(M_X) = I_2 (discrete trace values)

**Example 3: Derived quotient singularity C^2 / Z_2**

X = [C^2 / Z_2] where Z_2 acts by -1. The cotangent complex has pi_1(L_X) = C at the origin.

- K_0(M_X) = Z^2 (two irreducible representations of Z_2)
- HC^0(M_X) = C^2 (invariant functions)
- HH_0(O_X) = C^2 + C (invariants + correction at fixed point)
- H^0_{dR}(X) = C^2
- Correction: C_X = C (correction at the fixed point of Z_2 action)
- Type(M_X) = I_2 (discrete trace values)
