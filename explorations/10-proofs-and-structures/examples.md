## 1. Derived Affine Space A^n_R

### Structure

The derived affine space A^n_R = Spec(R[x_1, ..., x_n]) where R is a commutative ring (typically R = C). The structure sheaf is O_{A^n_R} = R[x_1, ..., x_n]. The cotangent complex L_{A^n_R} is the free R-module generated by dx_1, ..., dx_n in degree 0.

### Derived K-theory K_*(M_{A^n_R})

**K_0:** By Quillen's localization theorem (F4), K_0(A^n_R) = K_0(R). For R = C, K_0(A^n_C) = Z generated by the structure sheaf O. For R = Z, K_0(A^n_Z) = Z.

**K_n for n > 0:** By Bott periodicity (F55), K_{n+2}(A^n_R) = K_n(A^n_R). The higher K-groups are:
- K_{2k}(A^n_R) = K_0(R) = Z for R = C
- K_{2k+1}(A^n_R) = K_1(R) = 0 for R = C

**K_*(A^n_C) = Z for even *, 0 for odd *.**

### Derived Cyclic Cohomology HC^*(M_{A^n_R})

**HC^0:** By the HKR theorem (F12), HC^0(A^n_R) = HH^0(A^n_R) = R. For R = C, HC^0(A^n_C) = C.

**HC^n:** By periodicity, HC^{n+2}(A^n_R) = HC^n(A^n_R). For R = C:
- HC^{even}(A^n_C) = C
- HC^{odd}(A^n_C) = C

### Derived Hochschild Homology HH_*(O_{A^n_R})

**HH_n:** By HKR (F12), HH_n(A^n_R) = Omega^n_{A^n_R/R}. The differential forms Omega^n_{A^n_R/R} are the free R-modules generated by dx_{i_1} wedge ... wedge dx_{i_n}.

**Explicit computation:**
- HH_0(A^n_R) = R (constants)
- HH_1(A^n_R) = R^n (generated by dx_i)
- HH_2(A^n_R) = R^{n choose 2} (generated by dx_i wedge dx_j)
- HH_k(A^n_R) = R^{n choose k}
- HH_n(A^n_R) = R (generated by dx_1 wedge ... wedge dx_n)
- HH_k(A^n_R) = 0 for k > n

**For R = C:** HH_k(A^n_C) = C^{n choose k}.

### Derived de Rham Cohomology H^*_{dR}(A^n_C)

**H^n_{dR}:** By the Poincare lemma for derived affine space, H^n_{dR}(A^n_C) = Omega^n_{A^n_C/C}.

**Explicit computation:**
- H^0_{dR}(A^n_C) = C (constants)
- H^1_{dR}(A^n_C) = C^n (generated by dx_i)
- H^2_{dR}(A^n_C) = C^{n choose 2}
- H^k_{dR}(A^n_C) = C^{n choose k}
- H^n_{dR}(A^n_C) = C
- H^k_{dR}(A^n_C) = 0 for k > n

**HKR isomorphism:** HH_k(A^n_C) = H^k_{dR}(A^n_C) for all k.

### Derived von Neumann Algebra M_{A^n_R}

**M_{A^n_R} = B(L^2(A^n_R)):** The derived von Neumann algebra is the algebra of bounded operators on the derived Hilbert module L^2(A^n_R) = L^2(R[x_1, ..., x_n]).

**Delta_{A^n_R} = exp(2 pi H):** The modular operator is the exponential of 2 pi times the Hamiltonian H = log(Delta_{A^n_R}).

**J_{A^n_R}:** The modular conjugation is the anti-unitary involution J_{A^n_R}(xi) = xi^* on L^2(A^n_R).

**sigma_t^{A^n_R}(a) = Delta_{A^n_R}^{it} a Delta_{A^n_R}^{-it}:** The modular automorphism group acts by conjugation with Delta_{A^n_R}^{it}.

## 2. Derived Elliptic Curve E

### Structure

The derived elliptic curve E over C is a derived stack with classical truncation E_0 = E (an elliptic curve over C) and derived structure given by the cotangent complex L_E. The tangent complex T_E has rank 1 in degree 0 and rank 1 in degree 1 (from the derived thickening). The canonical bundle omega_E = O_E (E is Calabi-Yau).

### Derived K-theory K_*(M_E)

**K_0:** By (F5), K_0(E) = Z x Pic(E) where Pic(E) = E(C) is the elliptic curve itself. The Z factor is the rank and E(C) is the Picard group.

**Explicit K_0:** K_0(E) = Z x E(C). The group structure is (n, L) + (m, M) = (n + m, L tensor M) where n, m in Z and L, M in E(C).

**K_n for n > 0:** By Bott periodicity, K_n(E) = K_n(C). For R = C:
- K_{2k}(E) = Z
- K_{2k+1}(E) = 0

### Derived Cyclic Cohomology HC^*(M_E)

**HC^0:** By the HKR theorem, HC^0(E) = HH^0(E) = C.

**HC^n:** By periodicity, HC^{n+2}(E) = HC^n(E). For R = C:
- HC^{even}(E) = C
- HC^{odd}(E) = C

**HC^*(E) = C[x, x^{-1}] where deg(x) = 2.**

### Derived Hochschild Homology HH_*(O_E)

**HH_n:** By HKR (F12), HH_n(E) = Omega^n_E. The differential forms Omega^n_E are generated by dx where x is a local coordinate on E.

**Explicit computation:**
- HH_0(E) = C (constants)
- HH_1(E) = C (generated by dx)
- HH_n(E) = C for n = 0, 1
- HH_n(E) = 0 for n > 1

### Derived de Rham Cohomology H^*_{dR}(E)

**H^n_{dR}:** By the Poincare lemma, H^n_{dR}(E) = Omega^n_E for n = 0, 1.

**Explicit computation:**
- H^0_{dR}(E) = C (constants)
- H^1_{dR}(E) = C (generated by dx)
- H^n_{dR}(E) = 0 for n > 1

**HKR isomorphism:** HH_n(E) = H^n_{dR}(E) for n = 0, 1.

### Derived von Neumann Algebra M_E

**M_E = B(L^2(E)):** The derived von Neumann algebra is the algebra of bounded operators on L^2(E) = L^2(E, O_E).

**Delta_E = exp(2 pi H):** The modular operator is exp(2 pi H) where H = log(Delta_E).

**Type(M_E):** Since K_0(E) = Z x E(C) has bounded trace (the rank part is Z and the Picard part is compact), M_E is Type II_1.

## 3. Derived Flag Variety F

### Structure

The derived flag variety F is the derived stack of complete flags in C^n. The classical truncation F_0 is the flag variety of complete flags C^k_1 subset C^k_2 subset ... subset C^k_n = C^n where 0 < k_1 < k_2 < ... < k_n = n. The derived structure is given by the cotangent complex L_F. The flag variety has dimension n(n-1)/2.

### Derived K-theory K_*(M_F)

**K_0:** By the Borel-Weil-Bott theorem extended to derived flag varieties, K_0(F) = Z^r where r is the number of Schubert cells. For the complete flag variety in C^n, r = n! (the number of permutations in S_n).

**Explicit K_0:** K_0(F) = Z^{n!}. The generators are the structure sheaves O_{X_w} of the Schubert varieties X_w indexed by w in S_n.

**K_n for n > 0:** By Bott periodicity, K_n(F) = K_n(Z). For R = Z:
- K_{2k}(F) = Z^{n!}
- K_{2k+1}(F) = 0

### Derived Cyclic Cohomology HC^*(M_F)

**HC^0:** By the HKR theorem, HC^0(F) = HH^0(F) = Z^{n!}.

**HC^n:** By periodicity, HC^{n+2}(F) = HC^n(F). For R = Z:
- HC^{even}(F) = Z^{n!}
- HC^{odd}(F) = Z^{n!}

### Derived Hochschild Homology HH_*(O_F)

**HH_n:** By HKR (F12), HH_n(F) = Omega^n_F. The differential forms Omega^n_F are generated by the Schubert cell decomposition.

**Explicit computation:**
- HH_0(F) = Z^{n!} (constants)
- HH_1(F) = Z^{n! * dim(F)} (generated by dx_i for each Schubert cell)
- HH_k(F) = Z^{n! * dim(F) choose k}
- HH_{dim(F)}(F) = Z^{n!}
- HH_n(F) = 0 for n > dim(F) = n(n-1)/2

### Derived de Rham Cohomology H^*_{dR}(F)

**H^n_{dR}:** By the Poincare lemma, H^n_{dR}(F) = Omega^n_F for n = 0, 1, ..., dim(F).

**Explicit computation:**
- H^0_{dR}(F) = Z^{n!}
- H^1_{dR}(F) = Z^{n! * dim(F)}
- H^n_{dR}(F) = Z^{n! * dim(F) choose n}
- H^{dim(F)}_{dR}(F) = Z^{n!}
- H^n_{dR}(F) = 0 for n > dim(F)

**HKR isomorphism:** HH_n(F) = H^n_{dR}(F) for all n.

### Derived von Neumann Algebra M_F

**M_F = B(L^2(F)):** The derived von Neumann algebra is the algebra of bounded operators on L^2(F) = L^2(F, O_F).

**Delta_F = exp(2 pi H):** The modular operator is exp(2 pi H) where H = log(Delta_F).

**Type(M_F):** Since K_0(F) = Z^{n!} has discrete trace values, M_F is Type I_{n!}.

## 4. Derived Projective Space P^n

### Structure

The derived projective space P^n = Proj(Sym(V^*)) where V = C^{n+1}. The structure sheaf is O_{P^n} = Proj(Sym(V^*)). The tangent complex T_{P^n} has rank n in degree 0 and rank 1 in degree 1. The canonical bundle omega_{P^n} = O_{P^n}(-n-1) is not trivial, so P^n is not Calabi-Yau.

### Derived K-theory K_*(M_{P^n})

**K_0:** By the Grothendieck-Riemann-Roch theorem, K_0(P^n) = Z^{n+1}. The generators are the structure sheaves O(k) for k = 0, 1, ..., n.

**Explicit K_0:** K_0(P^n) = Z^{n+1}. The relation is given by the Euler sequence: 0 -> O -> O(1)^{n+1} -> T_{P^n} -> 0.

**K_n for n > 0:** By Bott periodicity, K_n(P^n) = K_n(Z). For R = Z:
- K_{2k}(P^n) = Z^{n+1}
- K_{2k+1}(P^n) = 0

### Derived Cyclic Cohomology HC^*(M_{P^n})

**HC^0:** By the HKR theorem, HC^0(P^n) = HH^0(P^n) = Z^{n+1}.

**HC^n:** By periodicity, HC^{n+2}(P^n) = HC^n(P^n). For R = Z:
- HC^{even}(P^n) = Z^{n+1}
- HC^{odd}(P^n) = Z^{n+1}

### Derived Hochschild Homology HH_*(O_{P^n})

**HH_n:** By HKR (F12), HH_n(P^n) = Omega^n_{P^n}. The differential forms Omega^n_{P^n} are generated by the Euler sequence.

**Explicit computation:**
- HH_0(P^n) = Z^{n+1} (constants)
- HH_1(P^n) = Z^{n+1} (generated by dx_i)
- HH_k(P^n) = Z^{n+1 choose k}
- HH_n(P^n) = Z^{n+1}
- HH_n(P^n) = 0 for n > n

### Derived de Rham Cohomology H^*_{dR}(P^n)

**H^n_{dR}:** By the Poincare lemma, H^n_{dR}(P^n) = Omega^n_{P^n} for n = 0, 1, ..., n.

**Explicit computation:**
- H^0_{dR}(P^n) = Z^{n+1}
- H^1_{dR}(P^n) = Z^{n+1}
- H^n_{dR}(P^n) = Z^{n+1 choose n}
- H^n_{dR}(P^n) = 0 for n > n

**HKR isomorphism:** HH_n(P^n) = H^n_{dR}(P^n) for all n.

### Derived von Neumann Algebra M_{P^n}

**M_{P^n} = B(L^2(P^n)):** The derived von Neumann algebra is the algebra of bounded operators on L^2(P^n) = L^2(P^n, O_{P^n}).

**Delta_{P^n} = exp(2 pi H):** The modular operator is exp(2 pi H) where H = log(Delta_{P^n}).

**Type(M_{P^n}):** Since K_0(P^n) = Z^{n+1} has discrete trace values, M_{P^n} is Type I_{n+1}.

### Index Computation for P^n

**Ind(D_{P^n}):** By the Atiyah-Singer index theorem, Ind(D_{P^n}) = int_{P^n} ch(D_{P^n}) td(T_{P^n}). The Chern character ch(D_{P^n}) = ch(O_{P^n}) = 1. The Todd class td(T_{P^n}) = (1 + (n+1)/2 c_1 + ...).

**E12 (spinor index):** E12 = hat{A}(P^n) ch(O_{P^n}) sqrt(hat{A}(T_{P^n})) = hat{A}(P^n) sqrt(hat{A}(T_{P^n})) because ch(O_{P^n}) = 1.

**Ratio:** Ind(D_{P^n}) / E12 = ch(O_{P^n}) / sqrt(hat{A}(T_{P^n})) = 1 / sqrt(hat{A}(T_{P^n})) because P^n is not Calabi-Yau.

### Key Observations

1. **K_0 determines type:** For all four examples, the trace on K_0 determines the type classification. A^n_R and P^n are Type I, E is Type II_1, and F is Type I_{n!}.

2. **HKR isomorphism holds:** For all four examples, HH_n = H^n_{dR} via the HKR isomorphism.

3. **Cyclic cohomology determines K-theory:** For all four examples, HC^* determines K_* via the Chern character.

4. **Index equals spinor index for Calabi-Yau:** E is Calabi-Yau so Ind = E12. A^n_R is Calabi-Yau so Ind = E12. F and P^n are not Calabi-Yau so Ind = E12 / sqrt(hat{A}).

5. **Bott periodicity:** All four examples satisfy K_{n+2} = K_n for all n.
