# Mission Statement — Phase 3 Agent 8

## Objective

Extend the Derived Modular Spectrum (DMS) framework to six interconnected domains that complete the theoretical architecture:

1. **Non-convex cones in non-rational fans** — Extend the cone-counting formula to non-convex cones, computing the correction term and showing how the fan determines intersection cohomology.
2. **Perfectoid schemes with sheaf trace** — Extend the p-adic trace formula from spaces to schemes where Frobenius kernel is a sheaf, computing S_p for specific perfectoid examples.
3. **Kähler manifolds with non-constant torsion** — Prove the extended Einstein equation simplifies for Kähler manifolds where Ric^{(2,0)+(0,2)} = 0.
4. **Product manifolds with non-constant torsion** — Compute Aut(M_{X_1 x X_2}) showing unitary group decomposition as a product.
5. **p-adic modular flow as a function of p** — Define sigma_t^{(p)} precisely as a function of the prime p, compute the p-adic L-function, and study the limit p -> infinity.
6. **Spectral triple for non-constant torsion** — Define (A, H, D) precisely and prove Delta_X = exp(D^2) for non-constant torsion with full DT^C.

## Success Criteria

- All 6 parts contain rigorous proofs, not just computations
- Each part has at least 5 theorems labeled PROVEN or CONJECTURED
- Intersection cohomology is computed for non-convex cone examples
- Sheaf trace formula is derived for perfectoid schemes
- Extended Einstein equation is proved for Kähler manifolds
- Product decomposition of Aut(M_X) is explicit
- p-adic L-function is defined and related to zeta function
- Spectral triple is defined with proof Delta_X = exp(D^2)
- At least 10 diagrams across all files
- All references verified against Agent 7's work

## What This Extends

Agent 7 established the cone-counting formula for non-rational fans with strongly convex cones. This agent extends to non-convex cones (removing the strong convexity requirement). Agent 7 computed p-adic entropy for perfectoid spaces; this agent extends to perfectoid schemes where the Frobenius kernel is a sheaf. Agent 7 proved the extended Einstein equation for general non-Kähler manifolds; this agent proves the Kähler case where Ric^{(2,0)+(0,2)} = 0. Agent 7 computed the modular group for individual manifolds; this agent computes the product decomposition. Agent 7 established the p-adic convergence condition; this agent studies the functional dependence on p. Agent 7 introduced the spectral triple; this agent proves Delta_X = exp(D^2).

## What Success Looks Like

A complete theoretical framework where the DMS program has addressed all 6 open questions from Agent 7's notes. The resulting body of work should be self-contained and rigorous enough to serve as the foundation for Phase 4, which will focus on synthesis and applications.
